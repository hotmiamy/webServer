This is when you finally understand why a URL starts with HTTP

Um servidor web é um programa que responde a solicitações HTTP (Hypertext Transfer Protocol) de clientes, como navegadores da web, e entrega páginas da web para esses clientes através da internet. Quando você digita um URL (Uniform Resource Locator) em seu navegador, o navegador envia uma solicitação para o servidor web especificado no URL. O servidor web então processa essa solicitação, localiza o recurso solicitado (geralmente um arquivo HTML, imagem, vídeo, etc.), e envia uma resposta de volta para o navegador, que exibe o conteúdo para você.
