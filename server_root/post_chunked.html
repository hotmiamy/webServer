Teste de Método POST com Encoding Chunked

Bem para Testar isso você tera q usar o terminal, digite o seguinte código nele:

"curl -X POST --data-binary "Texto ou arquivo" --header "Transfer-Encoding: chunked" --header "Content-Type: " http://localhost:8080/post_files/"

Ps: para mandar arquivo colocar '@' no começo; se não mandar Content-Type ele vai definir como arquivo comum sem extenção.
